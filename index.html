WebGL Geospatial Visualization Lab

The mountains and depths of Earth, in 3D

A Game of Thrones-style cinematic sweep over the real planet — terrain and ocean trenches dramatically exaggerated. Three rendering engines, the same experience, so you can feel the trade-offs side by side.

⚠️ Your browser/device doesn't appear to support WebGL, which all three prototypes require. Try a recent desktop Chrome, Firefox, Safari, or Edge.
Data-accurate

CesiumJS

Real global elevation and ocean-floor bathymetry (GEBCO-based), with runtime vertical exaggeration. The most faithful "real Earth" view.

● Ocean depths: native Explore →
Stylized

Three.js

A displacement-mapped globe where the geometry itself deforms from a height map — the closest to a hand-built "fantasy map" look and feel.

● Ocean depths: artistic Explore →
Free & open

MapLibre GL JS

An open-source globe with 3D land terrain and no vendor lock-in. The honest baseline — great relief on land, but oceans render flat.

● Ocean depths: none Explore →

Each prototype plays a cinematic intro on load, then hands control to you (orbit, zoom, tilt). Use the Relief slider to exaggerate the terrain in real time. For the full experience, add free API keys to .env — see .env.example. Everything still runs without them via graceful fallbacks.
